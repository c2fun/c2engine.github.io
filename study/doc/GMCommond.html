常用传送地点

传送命令基本格式：/teleport mapid x z y 
 mapid = 地图ID
 x = 横坐标
 z = 纵坐标
 y = 高度坐标
 世界地图ID统一为11000

/teleport 11000 1989 2894 130 ——狼出生地
/teleport 11000 1650 2920 40 ——人类出生地
/teleport 11000 1639 2683 88 ——吸血鬼出生地
/teleport 11000 2844 2444 100 ——沙漠
/teleport 11000 2999 2143 70 ——乌云墓地
/teleport 11000 2467 2285 45 ——赏金镇
/teleport 11000 2387 2489 50 ——斯提兰岛
/teleport 11000 2251 2350 80 ——红叶石林
/teleport 11000 2455 1943 65 ——克伦尔考古营地
/teleport 11000 1954 2251 110 ——兰德格莉德镇
/teleport 11000 1680 2405 88 ——矿工营地
/teleport 11000 1599 2488 75 ——矿洞

副本传送命令： /teleportgroup ID 
 ID=副本地图ID

/teleportgroup 1 ——阿西尔神殿遗迹
/teleportgroup 2 ——斯通卡素王国
/teleportgroup 3 ——阿蒂亚古王国陵寝
/teleportgroup 4 ——暴风巢穴

常用命令

显示所有基本指令		/help												
重新载入技能基本信息	/reload skillinfo												
重新载入特效资源		/reload texiao												

添加宠物          /addpet mobID level              mobID = 宠物ID；level = 宠物等级				/addpet 2004 30		获得一只30级的小鱼人（ID2004）宠物		
宠物升级		/petlevelup level		level = 提升等级数值					/petlevelup 3		被召唤出来的宠物等级提升3级		
宠物学技能	/learnpetskill id level		id= 技能ID；level = 技能等级									

添加物品		/additem itemID count		itemID = 物品ID；count = 添加物品数量，不写默认为无限		/additem 40064 2	   获得2份幽蓝卷轴（ID40064）		
角色升级		/levelup add 			add = 提升的等级数，可以为负数					/levelup 3		   提升自己等级3级		
学习技能		/learnskill id level		id = 技能ID；level = 技能等级					/learnskill 102 1	   学习1级的急速之法（ID102)技能		
遗忘技能		/forgetskill index		index = 技能在技能栏的位置（游戏中只对种族技能有效）			/forgetskill 3		   遗忘技能天赋栏里面第三个技能(普通攻击不计入）		
瞬移移动		/teleport mapid x z y		详细说明见“1.常用传送地点”									
移动速度          /movespped count		        count = 移动具体数值					         /movespeed 5		   移动速度为5，可对其它玩家和NPOC使用		
复活玩家		/revive			

重置冷却时间(CD)(下面两条指令都要执行）
/luarenew 11 LookupOnlinePlayerByName("角色名字")._Actor:getCDManagerTable():clean()
/lua GetUser()._Actor:getCDManagerTable():clean()



学习专业技能	/learnpro pro level		pro = 专业ID，level = 熟练度；专业熟练度上限 = level+50		/learnpro 201 50	   增加武器锻造（ID201）技能熟练度50点		
学习配方		/learnrecipe pro recipe		pro = 专业ID；recipe = 配方id					/learnrecipe 201 30374	   学会打造武器冰雪之球（ID30374）

给角色圆币	/woyaoqian amount		amount = 圆币数量；单位：铜					/woyaoqian 5000		   给自己5000铜圆币（50银）		
环币             /luarenew 11 LookupOnlinePlayerByName("name")._Actor:addToken(amount);                           name = 玩家自己的名字        amount = 环币数量，单位：铜 
                  
踢掉玩家		/kick playername		playername = 玩家名称									
召唤玩家		/summon playername 												

消息广播         /broadcast word			word = 具体信息内容，默认消息刷新3次				/broadcast 欢迎来带暮光世界！ 游戏屏幕中央发布信息“欢迎来带暮光世界”		
显示在线人数     /playercount												

一击必杀          /kill			         立即杀死目标怪物、玩家、NPC									
直接杀死怪物	/killmonster ID count		ID = 怪物ID；count = 具体数量					/killmonster 1209 10		杀死10个已接任务中要求击杀的合卜萨（ID1209）		
任务完成		/taskcomplete id para		id = 任务ID；para为空，任务完成；para = 1，任务未完成		/taskcomplete 2215 （1）		ID为2215的任务已经完成（未完成）

让某角色禁言	/nospeaking  duration RoleName		duration = 禁言/冻结时长；单位：分                     	/nospeaking 5 XX		禁制XX在世界频道发言，禁言时间5分钟		
冻结某角色	/lockrole duration RoleName								        /lockrole 5 XX		        冻结游戏角色XX，冻结时间5分钟		
冻结某角色的账号	/lockaccount duration RoleName	        Rolename = 角色名称					/lockaccount 5 XX		冻结游戏角色XX所在的账号，冻结时间5分钟		

改变角色外观	/charlook id                    id = 外观ID,测试用，只对本地有效					/charlook 350		        角色外观改变为ID350的外观模型

宠物学技能

指令一 /luarenew 11 LookupOnlinePlayerByName("name"):learnPetSkill(1, 352, 1) 

name = 你的名字 							
参数一 —— 宠物在列表中的位置  （1~5）							
参数二 —— 技能id  （宠物技能id352~371）   							
参数三 —— 技能等级 (技能等级1~3)

指令二 /learnpetskill id level 

ID = 技能ID
level = 技能等级

灵力、怒气添加指令

增加角色灵力
/luarenew 11 LookupOnlinePlayerByName("name")._Actor:addSoulExp(数字)
/luarenew 11 LookupPlayerByDBID(DBID)._Actor:addSoulExp(数字)

name = 角色的名字						
DBID = 角色数字ID						
数字 = 给该角色的具体灵力值

加怒气，变身用
/luarenew 11 lua_buffer.addRageByID(DBID, 1000)

DBID = 角色数字ID						
1000 = 角色怒气最大值

给灵魂圣器或武器
/luarenew 11 LookupPlayerByDBID(DBID):addSoulEquip(ID,数量,等级, 0)

ID = 圣器或者普通武器的物品ID						
数量 = 需要该武器或圣器的数量						
等级 = 武器或圣器的灵魂等级

加buff
/luarenew 11 lua_buffer.luabuffertestfunction(PlayerDBID, BuffID, 持续时间毫秒, 0, 0, true)

巫会升级相关指令

/luarenew 11 Sociality.addGoldToGuildByPlayer(参数一, 参数二) 

 对工会仓库增加资金. 第一个参数是玩家的DBID, 第二个参数是增加的资金数.

/luarenew 11 Sociality.addGloryToGuildByPlayer(参数一, 参数二) 

 增加玩家个人贡献度. 第一个参数是玩家的DBID, 第二个参数是增加的贡献值.

/luarenew 11 Sociality.addHonorToGuildByPlayer(参数一, 参数二) 

 增加工会的荣誉值. 第一个参数是玩家的DBID, 第二个参数是增加的荣誉值.

城战开启指令

/luarenew 11 test_resetSiegeWar(1, true);

 ——只清空申请者者数据

/luarenew 11 test_resetSiegeWar(1, false);

 ——清空占领者/申请者数据

执行此指令前可以执行以下指令修改一些参数（参数说明见/common_config/siege_war/siege_war_config.lua）：
/luarenew 13 SiegeWar.SIEGE_PREPARE_DAYS = 0;

 ——城战申请开始后几天开战

/luarenew 13 SiegeWar.SIEGE_START_HOUR = 14;

 ——几点开始城战

/luarenew 13 SiegeWar.SIEGE_START_MINUTE = 0;

 ——多少分开始城战

/luarenew 11 SiegeWar.SIEGE_DURATION_TIME = 120;

 ——城战持续时间
